# Geronimo Stilton Character Homepage

A React + Vite single-page portfolio imagining Geronimo Stilton as a real-world editor, explorer, and storyteller with his own polished personal website.

## Character Description

Geronimo Stilton is the editor of *The Rodent's Gazette* in New Mouse City. Thoughtful, anxious, and unexpectedly brave, he turns every chaotic adventure into a story about curiosity, kindness, and courage. This homepage presents him as a living creative professional with featured adventures, personality traits, and an editorial world built around books, maps, and front-page discoveries.

## Run Locally

```bash
npm install
npm run dev
```

## Build For Production

```bash
npm run build
```
